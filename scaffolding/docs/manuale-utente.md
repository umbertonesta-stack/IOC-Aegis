# Manuale utente

> Scritto per chi **usa** il programma, non per chi lo ha scritto. Niente dettagli
> interni: solo come si installa, come si lancia, cosa fa ogni comando, con esempi.

## Installazione

```bash
git clone <URL>
cd <repo>
pip install -r requirements.txt
```

Richiede Python 3.11+.

## Uso

```bash
python -m <progetto> [opzioni] <argomenti>
```

### Comandi e opzioni

| comando / opzione | cosa fa | esempio |
|---|---|---|
| `--help` | mostra l'aiuto | `python -m <progetto> --help` |
| … | … | … |

### Esempi pratici
